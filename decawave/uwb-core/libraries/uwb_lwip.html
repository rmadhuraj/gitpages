uwb_lwip¶

  1. lwIP is a light-weight implementation of the TCP/IP protocol suite.
  2. Refer to the lwip_tx and lwip_rx for the usage of lwip apps.

API¶

lwip service

This is the lwip base class which utilizes the functions to do the configurations related to lwip layer based on dependencies.

paul kettle
